# 仿小米商城官网

> 一个基于 HTML/CSS/JavaScript 独立完成的静态商城页面，仿小米商城官网风格。

## 项目简介

本项目是仿照小米商城官网制作的静态前端页面，包含首页、商品展示区、轮播图等模块。项目完全由我独立完成，用于练习和巩固 HTML/CSS/JavaScript 前端基础。

## 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 | 页面结构 |
| CSS3 | 页面样式、Flex布局、响应式设计 |
| JavaScript | 交互效果（轮播图、选项卡等） |

## 页面功能

- 首页布局：头部导航、轮播图、商品分类展示区
- 商品展示：热门商品卡片网格布局
- 交互效果：轮播图自动播放 + 手动切换
